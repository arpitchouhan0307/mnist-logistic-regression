# Handwritten Digit Recognition using Logistic Regression

## Overview
This project implements handwritten digit classification using the MNIST dataset and Logistic Regression.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-Learn
- Matplotlib

## Workflow
1. Load MNIST dataset
2. Data preprocessing
3. Train-Test Split
4. StandardScaler
5. PCA (95% variance retained)
6. Logistic Regression Training
7. Model Evaluation

## Results
- Accuracy: 92.08%
- Macro F1 Score: 92.56%

## Evaluation Metrics
- Accuracy Score
- F1 Score
- Confusion Matrix
- Cross Validation

## Key Concepts
- Classification
- Dimensionality Reduction (PCA)
- Feature Scaling
- Logistic Regression
